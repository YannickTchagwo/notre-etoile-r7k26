# Bienvenue Ralph

Site de naissance de la famille Tchagwo, prêt pour GitHub Pages.

## Personnalisation

1. Remplacez les valeurs `XX.XX.XXXX`, `XX:XX`, `X,XX kg` et `XXX cm` dans `index.html`.
2. Placez la photo dans `assets/images/ralph.jpg`.
3. Remplacez le bloc `<div class="photo-placeholder">…</div>` par :
   `<img class="photo-placeholder" src="assets/images/ralph.jpg" alt="Ralph">`
4. Pour activer le compteur, ajoutez la date au format `AAAA-MM-JJ` dans `data-birth=""`.
5. Le bouton WhatsApp propose déjà d'écrire à maman ou à papa. Les numéros et le message se trouvent dans `script.js`.

## Publication

Créez un dépôt GitHub au nom difficile à deviner, déposez-y ces fichiers, puis activez **Settings → Pages → GitHub Actions**. Le workflow inclus publiera automatiquement le site.

## Confidentialité

Les balises `noindex` et `robots.txt` demandent aux moteurs de recherche de ne pas référencer le site. Cela ne constitue pas un contrôle d’accès : toute personne qui obtient l’adresse peut voir et retransmettre la page. Pour une vraie protection par mot de passe, utilisez plutôt Cloudflare Access, Netlify avec protection, ou un hébergement authentifié.
